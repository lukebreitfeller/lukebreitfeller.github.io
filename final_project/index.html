Events In The Future

This tool was built for use in human annotations for machine learning. Its aim is to take a text populated with event mentions (words or phrases used to signify events within the narrative) and give trained human users the ability to organize these events along a visual timeline. This process allows users to answer questions about the chronological order of any events in the text more clearly and efficiently. The end goal is to extract these timeline annotations for use in training machine learning models
